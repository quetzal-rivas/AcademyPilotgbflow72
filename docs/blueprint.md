# **App Name**: Academia Pilot

## Core Features:

- Landing Page Marketing: A responsive landing page presenting the core value proposition of AI-powered commercial automation for academies, including sections for features, how it works, and calls to action.
- Secure User Authentication: User registration and login system via email/password, allowing academy owners to create and access their personalized dashboard.
- AI Ad Campaign Tool: A tool within the dashboard enabling users to configure and launch advertising campaigns on Meta platforms, utilizing generative AI for creating optimized images and ad copy.
- Omnichannel AI Assistant Configuration: Set up and manage AI-driven customer service agents that respond automatically to inquiries on WhatsApp, Messenger, and SMS to capture and qualify leads.
- AI Voice Agent Lead Qualification Tool: A tool to configure an AI-powered voice agent that attends calls, qualifies leads, and collects basic information by integrating with voice APIs like ElevenLabs.
- API Key & Integration Management: A dedicated section in the dashboard for academy owners to securely add, update, and manage their API keys for Meta, Twilio, ElevenLabs, and preferred AI models, storing configurations via Firebase SDK.
- Automation Dashboard Overview: A central dashboard providing an at-a-glance overview of active marketing campaigns, conversation metrics, and lead generation from all AI automation features.
- Lead Management Dashboard: View, filter, and manage all current leads captured through AI automation and other channels, stored in the database.

## Style Guidelines:

- Primary color: A sophisticated and energetic deep blue (#4D62DE) to convey professionalism and innovation in automation.
- Background color: A dark, desaturated grey-blue (#17191C) providing a sleek, focused backdrop suitable for a high-tech platform.
- Accent color: A vibrant sky blue (#66CCFF) used to highlight interactive elements, calls to action, and important notifications.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, tech-inspired feel. Body font: 'Inter' (sans-serif) for clean readability and a neutral, objective tone.
- Utilize a consistent set of clean, minimalist vector icons with a preference for outline or glyph styles, aligning with the modern and efficient aesthetic of the application.
- Implement a modular, grid-based layout for the dashboard to ensure optimal organization and clear hierarchy of information, complemented by generous whitespace to enhance readability and user focus.
- Incorporate subtle, fast animations for UI interactions, such as button hovers, form field focus, and content transitions, to provide a responsive and polished user experience without being distracting.
